🧠 Exercícios - map, filter, reduce

1. (map - básico)

Crie um novo array com o dobro de cada número:

const numeros = [1, 2, 3, 4]

2. (map - objetos)

Crie um array contendo apenas os nomes:

const pessoas = [ { nome: "Ana", idade: 20 }, { nome: "João", idade: 25 }, { nome: "Carlos", idade: 30 } ]

3. (filter - básico)

Filtre apenas os números maiores que 20:

const numeros = [10, 15, 20, 25, 30]

4. (filter - objetos)

Retorne apenas os produtos com preço acima de 100:

const produtos = [ { nome: "Mouse", preco: 50 }, { nome: "Teclado", preco: 150 }, { nome: "Monitor", preco: 800 } ]

5. (reduce - soma)

Some todos os valores:

const numeros = [5, 10, 15]

6. (reduce - objetos)

Calcule o valor total da compra:

const carrinho = [ { produto: "Camisa", preco: 50 }, { produto: "Calça", preco: 100 }, { produto: "Tênis", preco: 200 } ]

7. (map + filter)

Retorne apenas os números pares multiplicados por 2:

const numeros = [1, 2, 3, 4, 5, 6]

8. (filter + reduce)

Some apenas os números maiores que 25:

const numeros = [10, 20, 30, 40, 50]

9. (reduce - contador)

Conte quantas vezes cada palavra aparece:

const palavras = ["maçã", "banana", "maçã", "laranja", "banana", "maçã"]

10. (nível HARD)

Retorne apenas alunos com nota ≥ 6 e calcule a média das notas deles:

const alunos = [ { nome: "Ana", nota: 7 }, { nome: "João", nota: 5 }, { nome: "Carlos", nota: 8 }, { nome: "Maria", nota: 4 } ]
