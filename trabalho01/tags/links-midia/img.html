Tag IMG

A tag <img> insere uma imagem na página. O atributo src especifica o caminho da imagem, e alt fornece uma descrição de texto para acessibilidade. É um elemento vazio (self-closing) e é essencial para conteúdo visual.

<img src="imagem.jpg" alt="Descrição da imagem">
<img src="./imagens/foto.png" alt="Foto de exemplo" width="300" height="200">
<img src="https://exemplo.com/img.gif" alt="Imagem externa">

Sempre inclua o atributo alt para descrever a imagem - é importante para SEO e acessibilidade. Use width e height para especificar dimensões, prevenindo layout shifts ao carregar.
