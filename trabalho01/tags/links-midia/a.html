Tag A

A tag <a> define um hiperlink que conecta documentos. O atributo href especifica a URL ou destino do link. É um dos elementos mais importantes da web, permitindo navegação entre páginas e seções. 

<a href="https://www.exemplo.com">Visite o Site</a>
<a href="./pagina-interna.html">Link Interno</a>
<a href="#secao-1">Ir para Seção</a>
<a href="mailto:email@exemplo.com">Enviar Email</a>

Use href para páginas externas (URLs completas), caminhos relativos para internes, #id para ancorar a seções, e mailto: para links de email. O atributo target="_blank" abre em nova aba.

Diferentes Tipos de Links

  • Link para página interna - Voltar ao início
  • Link externo - Abre em nova aba
  • Link de email - Para enviar mensagens
  • Link de âncora - Pula para seção da página

Os links são azuis e sublinhados por padrão, tornando-os facilmente identificáveis como navegáveis.
