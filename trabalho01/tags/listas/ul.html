Tag UL

A tag <ul> define uma lista não ordenada (com bullets/pontos). Cada item é definido com <li>. Use quando a ordem não é importante, como em listas de características ou itens.

<ul>
  <li>Maçã</li>
  <li>Banana</li>
  <li>Laranja</li>
</ul>

O tipo de marcador pode ser customizado com CSS usando list-style-type. Os valores podem ser disc (padrão), circle, square, ou uma imagem customizada.

Frutas Disponíveis
