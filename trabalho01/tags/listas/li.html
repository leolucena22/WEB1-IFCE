Tag LI

A tag <li> define um item de lista e deve estar sempre dentro de uma lista pai (<ul>, <ol>, ou <dl>). Cada <li> representa um item individual na lista.

<ul>
  <li>Primeiro item</li>
  <li>Segundo item</li>
  <li>Terceiro item</li>
</ul>

O <li> pode conter qualquer tipo de conteúdo: texto, imagens, links, parágrafos, etc. Sempre use <li> dentro de uma lista pai; não o utilize isoladamente.
