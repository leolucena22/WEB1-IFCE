Tag OL

A tag <ol> define uma lista ordenada (numerada). Cada item é definido com <li> e é automaticamente numerado. Use quando a ordem é importante, como em instruções ou rankings. 

<ol>
  <li>Passo um</li>
  <li>Passo dois</li>
  <li>Passo três</li>
</ol>

A tag <ol> suporta atributos como type (para mudar o tipo de numeração: 1, a, A, i, I) e start (para iniciar em um número diferente).

Exemplo Prático

Como Fazer um Bolo de Chocolate

  1. Pré-aqueça o forno a 180°C
  2. Misture farinha, açúcar e chocolate em pó
  3. Adicione ovos, leite e manteiga derretida
  4. Despeje a massa em uma forma untada
  5. Asse por 30 a 40 minutos
  6. Deixe esfriar antes de desenformar

Note como a lista ordenada automaticamente numera cada passo em sequência, perfeito para instruções e receitas.
