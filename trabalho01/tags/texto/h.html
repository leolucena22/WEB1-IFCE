Tag H1-H6

As tags <h1> até <h6> definem títulos e subtítulos com diferentes níveis de importância. O <h1> é o mais importante (geralmente usado uma vez por página), enquanto <h6> é o menos importante.

<h1>Título Principal</h1>
<h2>Subtítulo Secundário</h2>
<h3>Subtítulo Terciário</h3>
<h4>Subtítulo Quaternário</h4>
<h5>Subtítulo Quinto</h5>
<h6>Subtítulo Sexto</h6>

Usar a hierarquia correta de títulos é importante para SEO e acessibilidade. Leitores de tela usam essas tags para navegar. Não pule níveis (ex: não vá de h1 diretamente para h3).
