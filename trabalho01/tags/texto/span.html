Tag SPAN

A tag <span> é um contêiner genérico inline para agrupar elementos para estilização com CSS ou manipulação com JavaScript. Diferente de <div> que é um elemento de bloco, o <span> não quebra linhas.

<p>Este é um texto vermelho dentro de um parágrafo.</p>
<p>Use span para estilizar trechos específicos</span> de texto.</p>

<span> é frequentemente usado com CSS classes para aplicar estilos localizados em texto. Sem CSS ou JavaScript, não adiciona nenhum valor visual ou semântico.

Exemplo Prático

Este parágrafo tem texto em vermelho usando uma tag span com estilo inline.

Você também pode usar spans com fundo colorido para destacar trechos específicos dentro de texto.

A flexibilidade da tag span com múltiplos estilos torna-a perfeita para ajustes precisos.
