Tag HR

A tag <hr> cria uma linha horizontal que representa uma quebra temática entre conteúdos (como mudança de tópico ou seção). É um elemento vazio (self-closing) e é renderizada como uma linha horizontal na página.

<h2>Primeira Seção</h2>
<p>Conteúdo da primeira seção...</p>
<hr>
<h2>Segunda Seção</h2>
<p>Conteúdo da segunda seção...</p>

Use <hr> para separar logicamente tópicos diferentes. Você pode estilizá-la com CSS para alterar cor, espessura e estilo. Sempre prefira CSS em vez de múltiplos <hr> para criar espaçamento.
