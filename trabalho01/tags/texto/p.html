Tag P

A tag <p> define um parágrafo de texto. O navegador adiciona automaticamente espaço em branco antes e depois do parágrafo (margem). É a tag mais comum para agrupar texto em blocos semânticos e legíveis.

<p>Este é um parágrafo de texto.</p>
<p>Este é outro parágrafo.</p>
<p>Parágrafos separam ideias e facilitam a leitura.</p>

Cada <p> é um elemento de bloco, o que significa que começa em uma nova linha e ocupa toda a largura disponível. Não use <p> apenas para espaçamento; utilize CSS para isso.
