Tag HEAD

A tag <head> é usada para definir informações meta sobre o documento HTML. Ela não é exibida diretamente na página, mas contém elementos que influenciam a renderização e o comportamento da página, como o título, descrição, palavras-chave, e links para folhas de estilo e scripts. A estrutura básica de um documento HTML é a seguinte:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título da Página</title> <!-- Título da página, exibido na aba do navegador -->
  <meta name="description" content="Descrição da página para SEO">
  <meta name="keywords" content="HTML, CSS, JavaScript">
  <link rel="stylesheet" href="styles.css"> <!-- Link para folha de estilo -->
  <script src="script.js"> <!-- Link para script JavaScript -->
</head>
<body>
  Conteúdo da página
</body>
</html>

O conteúdo dentro da tag <head> é crucial para a otimização de mecanismos de busca (SEO) e para garantir que a página seja exibida corretamente em diferentes dispositivos e navegadores. O título da página, por exemplo, é importante para os usuários identificarem a página na aba do navegador e para os mecanismos de busca entenderem o conteúdo da página.

Exemplo Prático

Nota: A tag <head> não exibe conteúdo visível na página, mas seu efeito é claro:

  • O título desta página "Tag HEAD" aparece na aba do navegador - definido no <title>
  • A página está responsiva em diferentes tamanhos - graças ao <meta name="viewport">
  • O estilo visual está aplicado - pela <link rel="stylesheet" href="...">
  • Este script funciona - pelo <script src="...">

Inspecione a tag <head> desta página pressionando F12 para ver esses elementos em ação.
