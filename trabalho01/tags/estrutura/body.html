Tag BODY

A tag <body> envolve todo o conteúdo visível da página web. É aqui que ficam os parágrafos, imagens, links, tabelas e todos os elementos que o usuário vê no navegador. Apenas um <body> pode existir por página.

<body>
  <header>
    <h1>Título Principal</h1>
  </header>
  <main>
    <p>Conteúdo da página</p>
  </main>
  <footer>
    <p>Rodapé</p>
  </footer>
</body>

A estrutura semântica dentro do body deve usar tags como <header>, <main>, <section>, <article> e <footer> para melhor acessibilidade e SEO.

Exemplo Prático

Você está vendo este conteúdo agora porque está dentro da tag <body> desta página!

Tudo o que você vê na página - títulos, parágrafos, imagens, links, listas - está dentro do <body>. É o container para todo conteúdo visível.

A tag <header> no topo com navegação, o <section> com conteúdo, tudo faz parte do body. Esta é a estrutura semântica moderna do HTML5, tornando a página acessível e bem organizada.

Resumindo: Se você vê, o conteúdo está no <body>. Se você não vê (como CSS ou configurações), está no <head>.
