Tag HTML

A tag <html> é a raiz de um documento HTML. Ela envolve todo o conteúdo da página, incluindo as tags <head> e <body>. A estrutura básica de um documento HTML é a seguinte:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título da Página</title>
</head>
<body>
  Conteúdo da página
</body>
</html>

O atributo lang é usado para especificar o idioma do conteúdo da página, o que é importante para acessibilidade e SEO. No exemplo acima, o idioma é definido como português do Brasil (pt-br).
