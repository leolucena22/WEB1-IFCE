Tags de Estrutura

As tags de estrutura são usadas para definir a estrutura geral de uma página web. Elas incluem:

  • html

    A tag raiz de um documento HTML. Ela envolve todo o conteúdo da página.

    Ver mais

  • Head

    Contém metadados sobre o documento, como título, links para estilos e scripts.

    Ver mais

  • body

    Contém o conteúdo principal de um documento HTML.

Tags de Texto

As tags de texto são usadas para formatar e organizar o conteúdo textual em uma página web. Elas incluem:

  • h1 a h6

    Usadas para definir títulos e subtítulos, com h1 sendo o mais importante e h6 o menos importante.

    Ver mais

  • p

    Usada para definir um parágrafo de texto.

    Ver mais

  • span

    Usada para aplicar estilos a uma parte específica do texto dentro de um elemento.
